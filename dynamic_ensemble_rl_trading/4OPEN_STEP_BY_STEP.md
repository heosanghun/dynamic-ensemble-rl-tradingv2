# 4open.science 단계별 가이드

## 현재 화면에서 해야 할 일

### 1단계: GitHub 로그인
**"Login with GitHub to anonymize" 버튼을 클릭하세요.**

이 버튼은 페이지 중앙에 있는 큰 보라색/파란색 버튼입니다.

## 전체 프로세스

### Step 1: GitHub 로그인
1. "Login with GitHub to anonymize" 버튼 클릭
2. GitHub 인증 페이지로 이동
3. GitHub 계정으로 로그인
4. 4open.science에 권한 부여

### Step 2: 대시보드 접근
1. 로그인 후 대시보드로 이동
2. "Anonymize Repository" 또는 "New Repository" 버튼 클릭

### Step 3: 저장소 선택
1. GitHub 저장소 목록에서 선택:
   - `heosanghun/dynamic-ensemble-rl-trading` 선택
   - 또는 저장소 이름 입력

### Step 4: 익명화 설정
1. 익명화할 용어 목록 완성:
   - 사용자명, 이메일 등 개인 정보 제거
   - 저장소 이름을 익명 이름으로 변경 (선택사항)
   
2. 설정 완료 후 "Anonymize" 버튼 클릭

### Step 5: 익명 링크 받기
1. 익명화 완료 후 익명 링크 받기
2. 형식: `https://anonymous.4open.science/r/abc123xyz`
3. 링크 ID 복사

## 중요 사항

- GitHub 저장소는 이미 Private으로 설정되어 있음
- 4open.science는 GitHub 저장소를 직접 익명화하는 방식
- 압축 파일(`dynamic_ensemble_rl_trading.zip`)은 백업용

## 문제 해결

**GitHub 로그인이 안 될 때:**
- GitHub 계정이 활성화되어 있는지 확인
- 브라우저 팝업 차단 해제
- 다른 브라우저 시도

**저장소가 목록에 안 보일 때:**
- 저장소가 Private인지 확인 (Private 저장소도 선택 가능해야 함)
- GitHub 권한이 올바르게 부여되었는지 확인
- 저장소 이름을 직접 입력해보기
